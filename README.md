# Code & data of humanitarian landmine action reports. 

The purpose of this repository is to test NLP applications for knowledge extraction and understanding of the unstructured humanitarian data. 
The dataset contains humanitarian reports (PDFs), ontologies (concepts for defining humanitarian entity relations), and human annotations as ground-truth.

The dataset can be collected from CMAC (Cambodian Mine Action Centre) website and the Geneva International Centre for Humanitarian Demining (GICHD) website. They are online and available for the demining community. 
The rights and credits for the data belongs to creators of these reports.

License of the code is added to the repository.
